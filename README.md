# 🎓 نظام جداول التوقيت والامتحانات

نظام متكامل لإدارة جداول التوقيت الأسبوعية وجداول الامتحانات لكلية العلوم الاقتصادية والتجارية وعلوم التسيير.

## 📋 المميزات

- **📅 جدول التوقيت الأسبوعي** — إدارة الحصص حسب المستوى والقسم
- **📝 جدول الامتحانات** — الدور الأول والثاني
- **🏫 أشغال القاعات** — عرض فوري لإشغال القاعات والمدرجات
- **🔄 تعويض الحصص** — البحث عن أوقات فراغ القاعات
- **📦 الأرشيف** — حفظ السنوات الدراسية بصيغة JSON وExcel

## 🏗️ المكونات

- السنوات الدراسية (من 2025/2026 وما بعدها)
- المستويات (ليسانس + ماستر)
- الأقسام الخمسة
- التخصصات
- قائمة الأساتذة
- 51 قاعة ومدرج (48 قاعة + مدرج أ/ب/ج)

## 🚀 التشغيل

افتح `index.html` مباشرة في المتصفح — لا يحتاج خادم.

## 🛠️ التقنيات

- HTML5 + CSS3 + JavaScript (Vanilla)
- RTL / Arabic UI
- تصميم متجاوب
